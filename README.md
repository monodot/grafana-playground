# grafana-playground

Demos of doing things with Grafana and the LGTM stack (Loki, Grafana, Tempo, Mimir).

> [!CAUTION]
> Most of these demos are incomplete. They are skeleton setups, to make it easier to do some learning or exploration with the Grafana LGTM stack or Grafana Cloud. Please use at your own risk! Always verify before deploying to production.
